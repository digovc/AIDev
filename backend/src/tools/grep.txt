Searches for a regular expression pattern in project files using ripgrep (rg).

- Fast recursive search powered by ripgrep CLI.
- pattern: The regex pattern to search for (required).
- path: Directory to search in relative to project root (optional, defaults to project root).
- ignore: List of glob patterns to ignore (optional).
- caseSensitive: Boolean flag for case-sensitive search (optional, default false).
- maxResults: Maximum number of matches to return (optional, default 100).

Returns a list of matches, each with file path, line number, column number, and matching text.
